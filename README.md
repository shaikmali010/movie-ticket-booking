# Movie Ticket Booking System

## Overview
This is a console-based Movie Ticket Booking System built using Java and Object-Oriented Programming concepts. The application simulates the process of browsing movies, selecting seats, and booking tickets.

---

## Features
- View available movies
- Select seats for a show
- Book tickets
- Manage booking data
- Basic data handling using in-memory storage

---

## Tech Stack
- Java
- OOP (Encapsulation, Abstraction, Inheritance)
- Console-based application
